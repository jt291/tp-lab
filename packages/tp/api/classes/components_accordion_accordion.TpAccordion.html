    Class TpAccordion

    Dependency

    tp-details

    Slot

    • One or more <tp-details> elements to place in the accordion.

    tp-before-expand - Emitted when an details item is instructed to expand but before it is shown. Calling event.preventDefault() will prevent the item from expanding. event.detail.item will contain the expanding item. tp-expand - Emitted after an details item has been expanded. event.detail.item will contain the expanded item. tp-before-collapse - Emitted when an details item is instructed to collapse but before it is hidden. Calling event.preventDefault() will prevent the item from collapsing. event.detail.item will contain the collapsing item. tp-collapse - Emitted after an details item has been collapsed. event.detail.item will contain the collapsed item.

    • disconnectedCallback(): void

      Invoked when the component is removed from the document's DOM.
