    Installation

    pnpm add @tp-lab/asciidoc

    Usage

    Basic Conversion

    import { convert } from '@tp-lab/asciidoc';

    const asciidoc = `
    * Item 1
    * Item 2
    * Item 3
    `;

    const html = convert(asciidoc);
    console.log(html);

    Output:

    <ul>
      <li>Item 1</li>
      <li>Item 2</li>
      <li>Item 3</li>
    </ul>

    With Formatting

    import { convertFormatted } from '@tp-lab/asciidoc';

    const html = await convertFormatted(asciidoc);
    // Returns beautifully formatted HTML

    Collapsible Blocks

    [%collapsible%open]
. Click to toggle
* Hidden item 1
* Hidden item 2

    Output:

    <details open>
      <summary>Click to toggle</summary>
      <ul>
        <li>Hidden item 1</li>
        <li>Hidden item 2</li>
      </ul>
    </details>
