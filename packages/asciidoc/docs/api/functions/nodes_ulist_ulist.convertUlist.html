    Function convertUlist

    • convertUlist(node: List): string

      Converts an Asciidoctor unordered list (ulist) to semantic HTML.

      Parameters

      • node: List

        The Asciidoctor list node

      Returns string

      The HTML string representation of the unordered list

      Example

      // AsciiDoc: 
      // * Item 1
      // * Item 2
      convertUlist(node)
      // Returns:
      // <ul>
      //   <li>Item 1</li>
      //   <li>Item 2</li>
      // </ul>

      Example

      // AsciiDoc: 
      // [%collapsible%open]
      // . My List
      // * Item 1
      // * Item 2
      convertUlist(node)
      // Returns:
      // <details open>
      //   <summary>My List</summary>
      //   <ul>
      //     <li>Item 1</li>
      //     <li>Item 2</li>
      //   </ul>
      // </details>
