    Class Section

    Description

    Methods for managing sections of AsciiDoc content in a document.

    Example

    <pre>
      section = asciidoctor.Section.create()
      section.setTitle('Section 1')
      section.setId('sect1')
      section.getBlocks().length // 0
      section.getId() // "sect1"
      section.append(newBlock)
      section.getBlocks().length // 1
    </pre>

    assignCaption

    • assignCaption(value?: string, captionContext?: string): void

      Generate and assign caption to block if not already assigned.

      If the block has a title and a caption prefix is available for this block, then build a caption from this information, assign it a number and store it to the caption attribute on the block.

      If a caption has already been assigned to this block, do nothing.

      The parts of a complete caption are: <prefix> <number>. <title> This partial caption represents the part the precedes the title.

      Parameters

      • Optionalvalue: string

        the String caption to assign to this block or nil to use document attribute.

      • OptionalcaptionContext: string

        the String context to use when resolving caption-related attributes. If not provided, the name of the context for this block is used. Only certain contexts allow the caption to be looked up.

    findBy

    • findBy(
          selector: Selector | ((block: AbstractBlock) => string | boolean),
          block?: (block: AbstractBlock) => string | boolean,
      ): AbstractBlock[]

      Query for all descendant block-level nodes in the document tree that match the specified selector (context, style, id, and/or role). If a function block is given, it's used as an additional filter. If no selector or function block is supplied, all block-level nodes in the tree are returned.

      Parameters

      • selector: Selector | ((block: AbstractBlock) => string | boolean)
      • Optionalblock: (block: AbstractBlock) => string | boolean

      Returns AbstractBlock[]

      a list of block-level nodes that match the filter or an empty list if no matches are found

      Example

      doc.findBy({'context': 'section'});
      // => { level: 0, title: "Hello, AsciiDoc!", blocks: 0 }
      // => { level: 1, title: "First Section", blocks: 1 }

      doc.findBy({'context': 'section'}, function (section) { return section.getLevel() === 1; });
      // => { level: 1, title: "First Section", blocks: 1 }

      doc.findBy({'context': 'listing', 'style': 'source'});
      // => { context: :listing, content_model: :verbatim, style: "source", lines: 1 }
